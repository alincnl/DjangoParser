# DjangoParser

Парсер, реализованный с помощью фреймворка Django. Названия товаров и цена сохраняются в _catalog_list.csv_.

**Запуск проекта:**
```
.venv\Scripts\activate   
python manage.py runserver
```
**URL:** https://siam-shop.ru/catalog/tayskaya_kukhnya/
